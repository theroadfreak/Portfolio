Карактеристики

Ке пишува "!play " а потоа секоја песна што ја има на листа. Може да слуша за влез (неморате да copy/paste).Можете да ги зачувате вашите листи.

Направено во Python користејки "kivy" а е дадено во .exe формат со Pyinstaller подоцна сите фајлови се компресирани во installer што можете да го добиете. овде , или погледнете го проектот на gitHub овде.

Add Копчето ке додаде во листа.

Start Typing ке пишува "!play", името на песна, ке притисне enter за секоја песна на листата.

Save List ја зачувува листата.

Load From Saved чита зачувана листа.

Start Listening ке чека да копирате текст и кога притискате "Home" копче, ке го внесе копираното во листа.

Remove Last Item го брише последниот елемент.
