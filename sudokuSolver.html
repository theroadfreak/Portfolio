What is it

Features

A Sudoku puzzle solver made in python, working on a recursion based algorithm and Tkinter library for the GUI.

It was frozen with pyinstaller in to an .exe format witch you can download here or you can revew the code on gitHub here.

Solve button will try and solve the puzzle if there is more than 16 valid numbers.

Test Case will give you a puzzle with 1 solution.

Empty out grid will clear the grid.
