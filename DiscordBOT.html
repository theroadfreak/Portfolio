What is it

Features

It will type "!play " and then the every song name that you entered in the list. It can listen for your input (you dont have to copy/paste).You can save and load your lists.

It was made in Python using "kivy" and frozen to exe with Pyinstaller later all the needed files where compresed in a installer witch you can get here , or you can view the project on gitHub here but there is no code.

Add button will add the text you entered above to the list.

Start Typing will start to type "!play", the song name then press enter for each individual song on the list.

Save List saves the current list.

Load From Saved self explanatory.

Start Listening will wait for you to copy some text and press "Home" button, it will insert the copyed text in the list.

Remove Last Item will remove the last added item.
