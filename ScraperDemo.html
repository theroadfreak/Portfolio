What is it

Features

It's a WebScraper, it searches a local store website for soundbars and returns their names , price and picture in an html document, wich the script opens at the end. It basically displays the info in your web-browser.

I used "Beautifle soup" and "Selenium" together with "Headless Firefox", i'm not offering a download as .exe on this code becouse of issues with "Windows Defender" but you can revew the code on gitHub here.

It returns: all the soundbars that where found together with their names prices and pictures.
