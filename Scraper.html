What is it

Features

It's a WebScraper, it searches 2 websites for your choice of shoe name and returns their name and price.

I used "Beautiful soup" and "requests" as a GUI it uses the windows Console, i'm not offering a download as .exe on this code because of issues with "Windows Defender" but you can review the code on gitHub here.

Shoe Name: gives you an opportunity to enter your chosen shoe name.

It returns: all the articles that were found together with their prices.
